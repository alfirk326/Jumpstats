Jumpstats plugin for SourceMod
===============

This plugin calculates the time between throwing and jumping, so you can find best jump timing.

Because it was made for a 100 tickrate server, it needs to be modified for other tickrate server.


## List of compatible games
 * **Counter-Strike: Source**
 * **Counter-Strike: Global Offensive**
 * **And so on..**
